Dialog

Dialog component for displaying custom content inside a popup dialog

Supported modifiers: alignment.
Read more about modifiers here.

NOTES

Use alignment modifiers to control the dialog positon (top, bottom, centerV, centerH, etc... by default the dialog is align to center)

PROPS

namedescriptiontypedefault
visibleControl visibility of the dialogbool
onDismissdismiss callback for when clicking on the backgroundfunc
overlayBackgroundColorThe color of the overlay backgroundstringColors.rgba(Colors.dark10, 0.6)
widthThe dialog width (default: 90%)union'90%'
heightThe dialog height (default: 70%)union'70%'
animationConfigthe animation configuration to pass to the dialog (based on react-native-animatable, ex. {animation, duration, easing,..})object
containerStyleThe dialog container styleunion
